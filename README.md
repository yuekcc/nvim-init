# .nvim

简单的 neovim 配置，只包含常规的配置和 lazy、tree-sitter 插件。

windows 系统安装到 `%APPDATA%/Local/nvim` 目录；Linux 系统安装到 `~/.config/nvim`。

环境要求：git、curl、zig。

```sh
# 比如在 Linux 下安装
mkdir -p ~/.config
cd ~/.config
git clone https://github.com/yuekcc/nvim-init.git nvim
```
